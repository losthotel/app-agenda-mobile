### COMPONENTES
- react-native-calendars - biblioteca de calendário pronto
- FlatList - listagem de compromissos, selecionar uma data, etc
-  Modal - para adicionar ou editar compromissos 
- DatePimePicker - para as horas
- Persistencia dos dados para armazenar compromissos - AsyncStorage
- react-native-device-info para pegar o ID único do aparelho.
